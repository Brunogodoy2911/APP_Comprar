## 📌 Sobre o App

Já teve dificuldade em organizar suas compras?
O **Comprar** veio para solucionar este problema!

O app surgiu de uma necessidade real: minha mãe precisava organizar as
compras aqui de casa.\
Com meu pensamento de **Desenvolvedor**, decidi criar o **Comprar**, uma
solução prática e eficiente para o dia a dia.

------------------------------------------------------------------------

## 🚀 Funcionalidades

-   📝 Criar um item de compra
-   ✅ Marcar um item como concluído
-   🗑️ Excluir um item da lista, ou a lista inteira

## 🖥️ Como utilizar na máquina

Clone o repositório:

``` bash
git clone https://github.com/Brunogodoy2911/APP_Comprar.git
```

Acesse o diretório do projeto:

``` bash
cd APP_Comprar
```

Instale as dependências:

``` bash
npm install
```

Inicie o projeto:

``` bash
npx expo start
```
